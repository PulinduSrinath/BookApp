# 📚 React Native Book Library App

A simple yet elegant mobile app built with **React Native** that allows users to browse a list of books, filter by genre, view detailed information, and manage a wishlist. The UI includes smooth animations and a splash screen for a professional touch.

---

## ✨ Features

- 📖 Book listing with cover, title, author, and genre
- 🔍 Genre-based filtering
- 💖 Add/remove books from wishlist
- 📄 Book detail view with description
- 🎬 Smooth animated transitions
- 🧼 Minimalist and modern UI
- 🚀 Splash screen on startup
